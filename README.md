### Data Type
DAPR - Number of days included in the multiday precipitation total (MDPR)

SNOW - Snowfall

TAVG - Average Temperature.

PRCP - Precipitation

MDPR - Multiday precipitation total (use with DAPR and DWPR, if available)

SNWD - Snow depth
